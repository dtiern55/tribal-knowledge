# Tribal Knowledge — Claude Guidance

## Project
Fantasy Survivor league web app. Stack: FastAPI (backend), React (frontend), Supabase (Postgres + auth).

See `fantasy-survivor-design.md` for the full scoring system, token/advantage mechanics, and design principles.

## Running the backend
```bash
cd backend
source .venv/bin/activate
uvicorn app.main:app --reload
```
